# 入门

本指南以最少步骤帮助新物业从零进入日常运营。

## 快速开始

- 打开 Web 应用或安装 Android APK。
- 使用管理员邮箱和密码完成**首次设置**。
- 在**设置**中配置酒店资料、货币、税费和季节。
- 添加客房、员工账户并分配角色。
- 使用模块指南（酒店、餐厅、迷你超市）培训员工，并在上线前配置备份。
