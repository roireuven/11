## 21 种语言的完整界面

Web 应用程序 UI 支持 **21 种语言环境**：英语、西班牙语、法语、德语、日语、韩语、阿拉伯语、印地语、泰语、越南语、印度尼西亚语、土耳其语、俄语、意大利语、荷兰语、波兰语、希伯来语、老挝语、葡萄牙语（巴西）、中文（简体）和中文（繁体）。

### 在哪里更改语言

|屏幕|如何|
|--------|-----|
| **登录/设置** |标题中的语言下拉列表（登录前）|
| **登录后** |顶部栏区域设置选择器或菜单中的**本地化** |
| **设置** |应用程序语言部分 |

首选项保存在浏览器存储中 (`hotel_mgr_uiLocale`)。

### RTL（从右到左）

**阿拉伯语**和**希伯来语**为整个应用程序启用 RTL 布局。模态表单使用改进的对齐方式，因此标签和输入可以在 LTR 和 RTL 语言中正确读取。

## 首次设置（已翻译）

设置向导已完全本地化：

- 企业/酒店名称
- 系统标题文本
- 管理员用户名、电子邮件和密码字段
- 所有按钮和验证消息

设置后，酒店名称将被存储并显示在配置的应用程序标题中。

## 翻译后的情态形式

添加和编辑对话框已本地化为所有 21 种语言，包括：

- **维护** — 新票（房间、优先级、问题、注释）
- **发票** — 添加/编辑（客人、房间、日期、金额、付款状态）
- **库存** — 添加/编辑商品（名称、条形码、类别、数量、POS 可用性）
- **菜单项** — 添加/编辑（名称、图标、价格、类别、图像、库存链接）
- **商店商品** — 添加/编辑（名称、价格、类别、货架图标、条形码、库存）- **用户帐户** — 添加/编辑（姓名、电子邮件、密码、角色）

图像上传标签（“来自设备”、“或图像 URL”）遵循活动语言。

## 预订 → 新客人

创建**预订**时，如果客人尚未在目录中：

1. 在预订表上点击**+新客人**（或同等内容）。
2. 填写**新客人**模式（姓名、护照、国籍、出生日期、付款方式、联系方式、备注）。
3. 点击 **添加客人并返回** — 您将返回预订并选择新客人。

国籍选择器（搜索列表）也已翻译。
