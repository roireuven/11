# Lokalisasi

HotelRestaurantMini-MartManagement mendukung **21 bahasa antarmuka** dengan tata letak RTL opsional untuk Arab dan Ibrani.

## Locale yang didukung

| Code | Bahasa |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

Dokumentasi tersedia dalam semua bahasa UI aplikasi. Halaman yang belum diterjemahkan menaut ke sumber bahasa Inggris.

Untuk mengubah bahasa, RTL, dan lokasi file locale, lihat [English localization guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/localization).
