# Memulai

Panduan ini membawa properti baru dari nol ke operasi harian dengan langkah minimal.

## Mulai cepat

- Buka aplikasi web atau instal APK Android.
- Selesaikan **Penyiapan pertama kali** dengan email dan kata sandi admin.
- Konfigurasi profil hotel, mata uang, pajak, dan musim di **Pengaturan**.
- Tambahkan kamar, akun staf, dan tetapkan peran.
- Latih staf dengan panduan modul (hotel, restoran, mini-mart) dan siapkan cadangan sebelum go-live.

## Panduan lengkap

Untuk instruksi langkah demi langkah yang detail, lihat [English getting started guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/getting-started).
