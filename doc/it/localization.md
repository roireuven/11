# Localizzazione

HotelRestaurantMini-MartManagement supporta **21 lingue dell'interfaccia** con layout RTL opzionale per arabo ed ebraico.

## Lingue supportate

| Code | Lingua |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

La documentazione è disponibile in tutte le lingue UI dell'app. Le pagine non ancora tradotte collegano alla fonte inglese.

Per cambiare lingua, RTL e posizione file locale, consulta la [English localization guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/localization).
