# Nawigacja i UI

Aplikacja to SPA z górnym paskiem, paskiem bocznym (menu), główną treścią i dolną nawigacją mobilną.

## Dokumentacja w aplikacji

| Lokalizacja | Etykieta | Działanie |
|----------|-------|--------|
| **Górny pasek** | Dokumentacja | Otwiera wbudowaną stronę dokumentacji |
| **Menu hamburger (☰)** | Pomoc → Dokumentacja | Pierwsza sekcja u góry paska bocznego |
| **Dolny pasek** | Docs | Dolna nawigacja (mobilna) |

Szczegóły układu, tryb ciemny i wybór języka znajdziesz w [English navigation guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/navigation-and-ui).
