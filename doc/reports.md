# Reports

**Path:** Sidebar → **Reports**

Available to **Admin** and **Manager**. Manager lands here by default after login.

## Dashboard analytics

The **Dashboard** (all roles with access) includes:

- Date range picker (from / to)
- Room availability vs occupied chart
- Bookings and invoices for selected dates
- Daily overview table

## Reports module

Manager-focused reporting includes:

- Sales by department: **Restaurant · Mini-Mart · Hotel (PMS) · POS**
- Occupancy and revenue trends
- Work period / shift context (POS / F&B work periods in data model)

## Work periods

POS and F&B **work periods (shifts)** support:

- Cash variance reporting
- Shift-bound transaction summaries

Exported in backup as `Work periods` table.

## Export for external analysis

Use **Settings → Export all data (CSV ZIP)** to analyze in Excel or BI tools.

Individual CSV buttons export one table at a time (rooms, bookings, transactions, restaurant, etc.).
