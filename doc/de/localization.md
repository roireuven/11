# Lokalisierung

HotelRestaurantMini-MartManagement unterstützt **21 Oberflächensprachen** mit optionaler RTL-Anordnung für Arabisch und Hebräisch.

## Unterstützte Sprachen

| Code | Sprache |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

Die Dokumentation ist in allen App-Oberflächensprachen verfügbar. Noch nicht übersetzte Seiten verlinken auf die englische Quelle.

Für Sprachwechsel, RTL und Speicherorte der Sprachdateien siehe die [English localization guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/localization).
