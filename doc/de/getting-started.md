# Erste Schritte

Diese Anleitung führt eine neue Immobilie mit minimalen Schritten vom Start bis zum Tagesbetrieb.

## Schnellstart

- Öffnen Sie die Web-App oder installieren Sie das Android-APK.
- Schließen Sie die **Ersteinrichtung** mit Admin-E-Mail und Passwort ab.
- Konfigurieren Sie Hotelprofil, Währung, Steuern und Saisons unter **Einstellungen**.
- Fügen Sie Zimmer, Mitarbeiterkonten hinzu und weisen Sie Rollen zu.
- Schulen Sie das Personal mit Modulhandbüchern (Hotel, Restaurant, Minimarkt) und richten Sie Sicherungen vor dem Go-live ein.

## Vollständige Anleitung
