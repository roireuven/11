# نظرة عامة

**HotelRestaurantMini-MartManagement** تطبيق إدارة موحد للمنشآت الفندقية الصغيرة والمتوسطة التي تجمع بين الفندق والمطعم والتجزئة في الموقع.

## الوحدات الأساسية

- **فندق / PMS** — الغرف، الحجوزات، تسجيل الدخول/الخروج، التدبير المنزلي، الصيانة
- **المطعم والمطبخ** — خدمة الطاولات، خدمة الغرف، طابور عرض المطبخ
- **ميني مارت ونقطة البيع** — مبيعات المتجر، رسوم الغرف، عمليات الصندوق
- **الإدارة** — الإعدادات، الحسابات، التقارير، النسخ الاحتياطي، التوطين

للحصول على نظرة عامة كاملة مع جداول الجمهور والقدرات، راجع [English overview](../en/overview.md).
