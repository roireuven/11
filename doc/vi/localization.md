# Bản địa hóa

HotelRestaurantMini-MartManagement hỗ trợ **21 ngôn ngữ giao diện** với bố cục RTL tùy chọn cho tiếng Ả Rập và Do Thái.

## Ngôn ngữ được hỗ trợ

| Code | Ngôn ngữ |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |

Tài liệu có sẵn bằng mọi ngôn ngữ UI của ứng dụng. Trang chưa dịch liên kết đến bản tiếng Anh.

Để đổi ngôn ngữ, RTL và vị trí tệp ngôn ngữ, xem [English localization guide](../en/localization.md).
