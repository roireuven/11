# ເລີ່ມຕົ້ນ

ຄູ່ມືນີ້ນຳທີ່ພັກໃໝ່ຈາກສູນໄປສູ່ການດຳເນີນງານປະຈຳວັນດ້ວຍຂັ້ນຕອນນ້ອຍທີ່ສຸດ.

## ເລີ່ມຕົ້ນຢ່າງວ່ອງໄວ

- ເປີດແອັບເວັບ ຫຼື ຕິດຕັ້ງ Android APK.
- ສຳເລັດ **ການຕັ້ງຄ່າຄັ້ງທຳອິດ** ດ້ວຍອີເມວ ແລະ ລະຫັດຜ່ານຜູ້ດູແລ.
- ຕັ້ງຄ່າໂປຣໄຟລ໌ໂຮງແຮມ, ສະກຸນເງິນ, ພາສີ ແລະ ລະດູການໃນ **ການຕັ້ງຄ່າ**.
- ເພີ່ມຫ້ອງ, ບັນຊີພະນັກງານ ແລະ ກຳນົດບົດບາດ.
- ຝຶກອົບຮົມພະນັກງານດ້ວຍຄູ່ມືໂມດູນ (ໂຮງແຮມ, ຮ້ານອາຫານ, ຮ້ານຂາຍເຄື່ອງ) ແລະ ຕັ້ງຄ່າສຳຮອງກ່ອນເປີດໃຊ້ງານຈິງ.

## ຄູ່ມືຄົບຖ້ວນ

ສຳລັບຄຳແນະນຳລະອຽດທຸກຂັ້ນຕອນ, ເບິ່ງ [English getting started guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/getting-started).
