# HotelRestaurantMini-MartManagement — ເອກະສານ

ເອກະສານຄົບຖ້ວນສຳລັບ **HotelRestaurantMini-MartManagement** (v2.0) — ລະບົບຈັດການທີ່ພັກຄົບວົງຈອນສຳລັບ **ໂຮງແຮມ**, **ຮ້ານອາຫານ**, **ຮ້ານຂາຍເຄື່ອງ** ແລະ **POS**.
> ບາງໜ້າມີແຕ່ພາສາອັງກິດ; ລິ້ງຊີ້ໄປທີ່ຮຸ່ນພາສາອັງກິດ.
