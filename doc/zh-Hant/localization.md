# 本地化

HotelRestaurantMini-MartManagement 支援 **21 種介面語言**，阿拉伯語與希伯來語可選 RTL 版面。

## 支援的語言

| Code | 語言 |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

文件提供所有應用介面語言版本。尚未翻譯的頁面連結至英文原文。

變更語言、RTL 及語言檔位置請參閱 [English localization guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/localization).
