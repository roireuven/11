## Полный интерфейс на 21 языке

Пользовательский интерфейс веб-приложения доступен в **21 языковом стандарте**: английском, испанском, французском, немецком, японском, корейском, арабском, хинди, тайском, вьетнамском, индонезийском, турецком, русском, итальянском, голландском, польском, иврите, лаосском, португальском (Бразилия), китайском (упрощенном) и китайском (традиционном).

### Где сменить язык

| Экран | Как |
|--------|-----|
| **Вход/настройка** | Раскрывающийся список языков в шапке (перед входом в систему) |
| **После входа** | Выбор локали в верхней панели или **Локализация** в меню |
| **Настройки** | Языковой раздел приложения |

Предпочтения сохраняются в памяти браузера (`hotel_mgr_uiLocale`).

### RTL (справа налево)

**Арабский** и **Иврит** позволяют использовать макет RTL для всего приложения. В модальных формах используется улучшенное выравнивание, поэтому метки и входные данные правильно читаются как на языках LTR, так и на языках RTL.

## Первоначальная настройка (переведено)

Мастер установки полностью локализован:

- Название компании/отеля
- Текст заголовка системы
- Поля имени пользователя администратора, электронной почты и пароля.
- Все кнопки и сообщения проверки

После настройки название отеля сохраняется и отображается в заголовке приложения, где оно настроено.

## Быстрые действия на панели управления (сетка PMS)

На **Панель управления** отображается сетка синих кнопок **+** для распространенных задач:

| Кнопка | Открывается |
|--------|--------|
| Добавить комнату | Новая форма помещения |
| Добавить бронирование | Новая форма бронирования |
| Добавить гостя | Новая гостевая форма |
| Добавить задачу | Новый талон на техническое обслуживание |
| Добавить услугу | Новый запрос на обслуживание |
| Добавить счет | Новая форма счета-фактуры |
| Добавить акции | Новый предмет инвентаря |
| Добавить меню | Новый пункт меню |
| Добавить товар в магазин | Новый магазин/мини-маркет |
| Добавить пользователя | Новый аккаунт сотрудников |

**Примечание.** *Добавить очистку* и *Добавить транзакцию* были удалены из этой сетки (версия 2.4). При необходимости используйте боковую панель для разделов **Управление** и **Транзакции**.

## Переведенные модальные формы

Диалоговые окна добавления и редактирования локализованы на всех 21 языке, включая:

- **Техническое обслуживание** — новый билет (комната, приоритет, выпуск, примечания)
- **Счет** — добавить/редактировать (гость, номер, даты, суммы, статус оплаты)
- **Инвентарь** — добавить/редактировать товар (название, штрих-код, категория, количество, наличие POS-терминала)
- **Пункт меню** — добавить/редактировать (название, иконку, цену, категорию, изображение, ссылку на сток)
- **Товар магазина** — добавить/редактировать (название, цену, категорию, значок полки, штрих-код, наличие)- **Учётная запись пользователя** — добавить/редактировать (имя, адрес электронной почты, пароль, роль)

Метки загрузки изображений («с устройства», «или URL-адрес изображения») следуют за активным языком.

## Бронирование → Новый гость

При создании **бронирования**, если гостя еще нет в каталоге:

1. Нажмите **+ Новый гость** (или аналогичный) в форме бронирования.
2. Заполните модальное окно **Новый гость** (имя, паспорт, национальность, дата рождения, способ оплаты, контакт, примечания).
3. Нажмите **Добавить гостя и вернуться** — вы вернетесь к бронированию с выбранным новым гостем.

Средство выбора национальности (список поиска) также переведено.
