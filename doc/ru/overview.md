# Обзор

**HotelRestaurantMini-MartManagement** — единое приложение управления для небольших и средних гостиничных объектов, сочетающих отель, ресторан и розничную торговлю на месте.

## Основные модули

- **Отель / PMS** — Номера, бронирования, заезд/выезд, уборка, обслуживание
- **Ресторан и кухня** — Обслуживание столов, room service, очередь на кухне
- **Минимаркет и POS** — Продажи в магазине, начисления на номер, касса
- **Администрирование** — Настройки, учётные записи, отчёты, резервное копирование, локализация

Полный обзор с таблицами аудитории и возможностей см. в [English overview](../en/overview.md).
