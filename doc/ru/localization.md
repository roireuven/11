# Локализация

HotelRestaurantMini-MartManagement поддерживает **21 язык интерфейса** с опциональной RTL-вёрсткой для арабского и иврита.

## Поддерживаемые языки

| Code | Язык |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

Документация доступна на всех языках UI приложения. Ещё не переведённые страницы ссылаются на английский оригинал.

Смена языка, RTL и расположение файлов локализации см. в [English localization guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/localization).
