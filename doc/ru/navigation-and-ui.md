# Навигация и интерфейс

Приложение — SPA с верхней панелью, боковой панелью (меню), основной областью и нижней навигацией на мобильных.

## Документация в приложении

| Расположение | Метка | Действие |
|----------|-------|--------|
| **Верхняя панель** | Документация | Открывает встроенную страницу документации |
| **Меню-гамбургер (☰)** | Справка → Документация | Первый раздел вверху боковой панели |
| **Нижняя панель** | Docs | Нижняя навигация (мобильная) |

Подробности макета, тёмный режим и выбор языка см. в [English navigation guide](../en/navigation-and-ui.md).
