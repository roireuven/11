# 시작하기

이 가이드는 새 시설을 최소 단계로 일상 운영까지 안내합니다.

## 빠른 시작

- 웹 앱을 열거나 Android APK를 설치하세요.
- 관리자 이메일과 비밀번호로 **최초 설정**을 완료하세요.
- **설정**에서 호텔 프로필, 통화, 세금, 시즌을 구성하세요.
- 객실, 직원 계정을 추가하고 역할을 할당하세요.
- 모듈 가이드(호텔, 레스토랑, 미니마트)로 직원을 교육하고 운영 전 백업을 설정하세요.
