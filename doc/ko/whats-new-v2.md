# v2.3 / v2.4의 새로운 기능

본 가이드에서는 HotelRestaurantMini-MartManagement의 **stable v2.3** 및 **stable v2.4**에 추가된 주요 기능을 요약하여 설명합니다.

**실시간 안정적인 사이트:**

| 버전 | URL |
|---------|-----|
| **v2.3** | [hotel-restaurant-minimart2-3.web.app](https://hotel-restaurant-minimart2-3.web.app/) |
| **v2.4** | [hotel-restaurant-minimart2-4.web.app](https://hotel-restaurant-minimart2-4.web.app/) |
| **개발** | [hotel-restaurant-minimart.web.app](https://hotel-restaurant-minimart.web.app/) |

---

## 21개 언어로 제공되는 전체 인터페이스

웹 앱 UI는 **21개 로케일**(한국어, 영어, 스페인어, 프랑스어, 독일어, 일본어, 한국어, 아랍어, 힌디어, 태국어, 베트남어, 인도네시아어, 터키어, 러시아어, 이탈리아어, 네덜란드어, 폴란드어, 히브리어, 라오어, 포르투갈어(브라질), 중국어(간체) 및 중국어(번체))로 제공됩니다.

### 언어를 변경하는 곳

| 화면 | 어떻게 |
|---------|-----|
| **로그인/설정** | 헤더의 언어 드롭다운(로그인 전) |
| **로그인 후** | 상단 표시줄 언어 선택기 또는 메뉴의 **현지화** |
| **설정** | 앱 언어 섹션 |

기본 설정은 브라우저 저장소(`hotel_mgr_uiLocale`)에 저장됩니다.

### RTL(오른쪽에서 왼쪽으로)

**아랍어** 및 **히브리어**는 전체 앱에 대해 RTL 레이아웃을 활성화합니다. 모달 양식은 향상된 정렬을 사용하므로 LTR 및 RTL 언어 모두에서 레이블과 입력이 올바르게 읽혀집니다.

---

## 최초 설정(번역됨)

설정 마법사가 완전히 현지화되었습니다.

- 업체/호텔 이름
- 시스템 헤더 텍스트
- 관리자 사용자 이름, 이메일 및 비밀번호 필드
- 모든 버튼 및 확인 메시지

설정 후에는 호텔 이름이 저장되고 구성된 앱 헤더에 표시됩니다.

---

## 대시보드 빠른 작업(PMS 그리드)

**대시보드**에는 일반적인 작업에 대한 파란색 **+** 버튼 그리드가 표시됩니다.

| 버튼 | 열림 |
|---------|---------|
| 방 추가 | 새로운 방 형태 |
| 예약 추가 | 새로운 예약 양식 |
| 게스트 추가 | 새로운 게스트 양식 |
| 작업 추가 | 새로운 유지보수 티켓 |
| 서비스 추가 | 신규 서비스 요청 |
| 송장 추가 | 새로운 송장 양식 |
| 재고 추가 | 새로운 인벤토리 아이템 |
| 메뉴 추가 | 새 메뉴 항목 |
| 상점 아이템 추가 | 신규 매장/미니마트 아이템 |
| 사용자 추가 | 새로운 직원 계정 |

**참고:** *정리 추가* 및 *트랜잭션 추가*가 이 그리드(v2.4)에서 제거되었습니다. 필요한 경우 **하우스키핑** 및 **거래**용 사이드바를 사용하세요.

---

## 번역된 모달 형식

추가 및 편집 대화 상자는 다음을 포함하여 21개 언어로 현지화되었습니다.

- **유지관리** — 새 티켓(객석, 우선순위, 문제, 메모)
- **청구서** — 추가/수정(투숙객, 객실, 날짜, 금액, 결제 상태)
- **인벤토리** — 품목 추가/수정(이름, 바코드, 카테고리, 수량, POS 가용성)
- **메뉴 항목** — 추가/수정(이름, 아이콘, 가격, 카테고리, 이미지, 주식 링크)
- **스토어 아이템** — 추가/수정(이름, 가격, 카테고리, 선반 아이콘, 바코드, 재고)- **사용자 계정** — 추가/수정(이름, 이메일, 비밀번호, 역할)

이미지 업로드 라벨('기기에서', '또는 이미지 URL')은 활성 언어를 따릅니다.

## 예약 → 신규 손님

**예약**을 생성할 때 게스트가 아직 디렉터리에 없는 경우:

1. 예약 양식에서 **+ 신규 고객**(또는 이에 상응하는 항목)을 탭합니다.
2. **새 게스트** 모달(이름, 여권, 국적, 생년월일, 결제 방법, 연락처, 메모)을 입력합니다.
3. **게스트 추가 및 복귀**를 탭하면 선택된 새 게스트와 함께 예약으로 돌아갑니다.

국적 선택기(검색 목록)도 번역됩니다.
