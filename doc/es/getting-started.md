# Primeros pasos

Esta guía lleva una nueva propiedad de cero a operaciones diarias con el mínimo de pasos.

## Inicio rápido

- Abra la aplicación web o instale el APK de Android.
- Complete la **Configuración inicial** con su correo y contraseña de administrador.
- Configure el perfil del hotel, moneda, impuestos y temporadas en **Ajustes**.
- Añada habitaciones, cuentas de personal y asigne roles.
- Capacite al personal con las guías de módulos (hotel, restaurante, minimarket) y configure copias de seguridad antes de entrar en producción.

## Guía completa

Para instrucciones detalladas paso a paso, consulte la [English getting started guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/getting-started).
