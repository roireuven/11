# Navegación e interfaz

La aplicación es una SPA con barra superior, barra lateral (menú), área de contenido principal y navegación inferior móvil.

## Documentación en la aplicación

| Ubicación | Etiqueta | Acción |
|----------|-------|--------|
| **Barra superior** | Documentación | Abre la página de documentación integrada |
| **Menú hamburguesa (☰)** | Ayuda → Documentación | Primera sección en la parte superior de la barra lateral |
| **Barra inferior** | Docs | Navegación inferior (móvil) |

Para detalles de diseño, modo oscuro y selector de idioma, consulte la [English navigation guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/navigation-and-ui).
