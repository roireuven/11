# Localización

HotelRestaurantMini-MartManagement admite **21 idiomas de interfaz** con diseño RTL opcional para árabe y hebreo.

## Idiomas admitidos

| Code | Idioma |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

La documentación está disponible en todos los idiomas de la interfaz. Las páginas aún no traducidas enlazan al original en inglés.

Para cambiar idioma, RTL y ubicación de archivos de idioma, consulte la [English localization guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/localization).
