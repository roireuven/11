# Primeiros passos

Este guia leva uma nova propriedade do zero às operações diárias com o mínimo de passos.

## Início rápido

- Abra o app web ou instale o APK Android.
- Conclua a **Configuração inicial** com e-mail e senha de administrador.
- Configure perfil do hotel, moeda, impostos e temporadas em **Configurações**.
- Adicione quartos, contas de funcionários e atribua funções.
- Treine a equipe com guias de módulos (hotel, restaurante, minimercado) e configure backup antes de entrar em produção.

## Guia completo

Para instruções detalhadas passo a passo, consulte o [English getting started guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/getting-started).
