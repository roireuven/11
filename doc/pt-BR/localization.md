# Localização

HotelRestaurantMini-MartManagement suporta **21 idiomas de interface** com layout RTL opcional para árabe e hebraico.

## Idiomas suportados

| Code | Idioma |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |

| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |

A documentação está disponível em todos os idiomas da interface. Páginas ainda não traduzidas linkam para a fonte em inglês.

Para alterar idioma, RTL e localização dos arquivos de locale, consulte o [English localization guide](../en/localization.md).
