# 概要

**HotelRestaurantMini-MartManagement** は、ホテル、レストラン、店内小売を組み合わせた中小規模の宿泊施設向け統合管理アプリケーションです。

## コアモジュール

- **ホテル / PMS** — 客室、予約、チェックイン/アウト、ハウスキーピング、メンテナンス
- **レストランとキッチン** — テーブルサービス、ルームサービス、キッチン表示キュー
- **ミニマートと POS** — 店舗販売、客室請求、レジ操作
- **管理** — 設定、アカウント、レポート、バックアップ、ローカライズ

対象者と機能表を含む完全な概要については、 [English overview](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/overview).
