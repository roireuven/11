# はじめに

このガイドは、新規施設を最小限の手順で日常運用まで導きます。

## クイックスタート

- Web アプリを開くか、Android APK をインストールします。
- 管理者メールとパスワードで **初回セットアップ** を完了します。
- **設定** でホテルプロフィール、通貨、税金、シーズンを構成します。
- 客室、スタッフアカウントを追加し、役割を割り当てます。
- モジュールガイド（ホテル、レストラン、ミニマート）でスタッフを研修し、本番前にバックアップを設定します。
