# ローカライズ

HotelRestaurantMini-MartManagement は **21 の UI 言語** をサポートし、アラビア語とヘブライ語では RTL レイアウトが利用できます。

## 対応ロケール

| Code | 言語 |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

ドキュメントはすべての UI 言語で利用できます。未翻訳のページは英語ソースにリンクします。

言語変更、RTL、ロケールファイルの場所については、 [English localization guide](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/localization).
