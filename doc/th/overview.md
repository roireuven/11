# ภาพรวม

**HotelRestaurantMini-MartManagement** เป็นแอปจัดการแบบรวมศูนย์สำหรับที่พักขนาดเล็กและกลางที่รวมโรงแรม ร้านอาหาร และร้านค้าในสถานที่

## โมดูลหลัก

- **โรงแรม / PMS** — ห้อง การจอง เช็คอิน/เอาต์ แม่บ้าน การบำรุงรักษา
- **ร้านอาหารและครัว** — บริการโต๊ะ รูมเซอร์วิส คิวแสดงครัว
- **มินิมาร์ทและ POS** — ขายร้านค้า คิดค่าห้อง การทำงานแคชเชียร์
- **การบริหาร** — การตั้งค่า บัญชี รายงาน สำรองข้อมูล การแปลภาษา

สำหรับภาพรวมฉบับสมบูรณ์พร้อมตารางผู้ใช้และความสามารถ ดู [English overview](../en/overview.md).
