# เริ่มต้นใช้งาน

คู่มือนี้พาทรัพย์สินใหม่จากศูนย์สู่การดำเนินงานประจำวันด้วยขั้นตอนน้อยที่สุด

## เริ่มต้นอย่างรวดเร็ว

- เปิดเว็บแอปหรือติดตั้ง Android APK
- ทำ **การตั้งค่าครั้งแรก** ด้วยอีเมลและรหัสผ่านผู้ดูแล
- กำหนดโปรไฟล์โรงแรม สกุลเงิน ภาษี และฤดูกาลใน **การตั้งค่า**
- เพิ่มห้อง บัญชีพนักงาน และกำหนดบทบาท
- ฝึกพนักงานด้วยคู่มือโมดูล (โรงแรม ร้านอาหาร มินิมาร์ท) และตั้งค่าสำรองก่อนเปิดใช้งานจริง

## คู่มือฉบับเต็ม
