## อินเทอร์เฟซเต็มรูปแบบใน 21 ภาษา

UI ของเว็บแอปมีให้บริการใน **21 ภาษา**: อังกฤษ, สเปน, ฝรั่งเศส, เยอรมัน, ญี่ปุ่น, เกาหลี, อารบิก, ฮินดี, ไทย, เวียดนาม, อินโดนีเซีย, ตุรกี, รัสเซีย, อิตาลี, ดัตช์, โปแลนด์, ฮิบรู, ลาว, โปรตุเกส (บราซิล), จีน (ตัวย่อ) และจีน (ดั้งเดิม)

### จะเปลี่ยนภาษาได้ที่ไหน

| หน้าจอ | อย่างไร |
|--------|-----|
| **เข้าสู่ระบบ/ตั้งค่า** | เมนูแบบเลื่อนลงภาษาในส่วนหัว (ก่อนลงชื่อเข้าใช้) |
| **หลังจากเข้าสู่ระบบ** | ตัวเลือกสถานที่ของแถบด้านบนหรือ **การแปลเป็นภาษาท้องถิ่น** ในเมนู |
| **การตั้งค่า** | ส่วนภาษาของแอป |

การตั้งค่าจะถูกบันทึกไว้ในที่เก็บข้อมูลของเบราว์เซอร์ (`hotel_mgr_uiLocale`)

### RTL (ขวาไปซ้าย)

**ภาษาอาหรับ** และ **ภาษาฮีบรู** เปิดใช้งานการจัดวาง RTL สำหรับทั้งแอป แบบฟอร์มโมดอลใช้การจัดตำแหน่งที่ได้รับการปรับปรุงเพื่อให้ป้ายชื่อและการป้อนข้อมูลอ่านได้อย่างถูกต้องทั้งในภาษา LTR และ RTL

## การตั้งค่าครั้งแรก (แปล)

วิซาร์ดการตั้งค่าได้รับการแปลเป็นภาษาท้องถิ่นอย่างสมบูรณ์:

- ชื่อธุรกิจ/โรงแรม
- ข้อความส่วนหัวของระบบ
- ช่องชื่อผู้ใช้ อีเมล และรหัสผ่านของผู้ดูแลระบบ
- ปุ่มทั้งหมดและข้อความตรวจสอบ

หลังจากตั้งค่า ชื่อโรงแรมจะถูกจัดเก็บและแสดงในส่วนหัวของแอปที่กำหนดค่าไว้

## การดำเนินการด่วนของแดชบอร์ด (ตาราง PMS)

**แดชบอร์ด** จะแสดงตารางปุ่ม **+** สีน้ำเงินสำหรับงานทั่วไป:

| ปุ่ม | เปิด |
|--------|--------|
| เพิ่มห้อง | ห้องพักรูปแบบใหม่ |
| เพิ่มการจอง | แบบฟอร์มการจองใหม่ |
| เพิ่มแขก | แบบฟอร์มแขกใหม่ |
| เพิ่มงาน | ตั๋วการบำรุงรักษาใหม่ |
| เพิ่มบริการ | คำขอบริการใหม่ |
| เพิ่มใบแจ้งหนี้ | แบบฟอร์มใบแจ้งหนี้ใหม่ |
| เพิ่มสต๊อก | รายการสินค้าคงคลังใหม่ |
| เพิ่มเมนู | รายการเมนูใหม่ |
| เพิ่มรายการร้านค้า | ร้านค้าใหม่ / มินิมาร์ทไอเท็ม |
| เพิ่มผู้ใช้ | บัญชีพนักงานใหม่ |

**หมายเหตุ:** *เพิ่มการทำความสะอาด* และ *เพิ่มธุรกรรม* ถูกลบออกจากตารางนี้ (v2.4) ใช้แถบด้านข้างสำหรับ **การดูแลทำความสะอาด** และ **ธุรกรรม** เมื่อจำเป็น

## แบบฟอร์มกิริยาที่แปล

กล่องโต้ตอบการเพิ่มและแก้ไขได้รับการแปลเป็นภาษาท้องถิ่นทั้งหมด 21 ภาษา ได้แก่:

- **การบำรุงรักษา** — ตั๋วใหม่ (ห้อง ลำดับความสำคัญ ปัญหา หมายเหตุ)
- **ใบแจ้งหนี้** — เพิ่ม/แก้ไข (แขก, ห้องพัก, วันที่, จำนวนเงิน, สถานะการชำระเงิน)
- **สินค้าคงคลัง** — เพิ่ม / แก้ไขรายการ (ชื่อ บาร์โค้ด หมวดหมู่ ปริมาณ ความพร้อมใช้งาน POS)
- **รายการเมนู** — เพิ่ม/แก้ไข (ชื่อ ไอคอน ราคา หมวดหมู่ รูปภาพ สต็อคลิงค์)
- **รายการร้านค้า** — เพิ่ม / แก้ไข (ชื่อ ราคา หมวดหมู่ ไอคอนชั้นวาง บาร์โค้ด หุ้น)- **บัญชีผู้ใช้** — เพิ่ม/แก้ไข (ชื่อ อีเมล รหัสผ่าน บทบาท)

ป้ายกำกับการอัปโหลดรูปภาพ (“จากอุปกรณ์”, “หรือ URL ของรูปภาพ”) เป็นไปตามภาษาที่ใช้งาน

## การจอง → แขกใหม่

เมื่อสร้าง **การจอง** หากแขกไม่ได้อยู่ในไดเรกทอรี:

1. แตะ **+ ลูกค้าใหม่** (หรือเทียบเท่า) บนแบบฟอร์มการจอง
2. กรอกโมดอล **แขกใหม่** (ชื่อ หนังสือเดินทาง สัญชาติ วันเดือนปีเกิด วิธีการชำระเงิน การติดต่อ บันทึก)
3. แตะ **เพิ่มแขก & กลับ** — คุณกลับไปที่การจองโดยเลือกแขกใหม่ไว้

เครื่องมือเลือกสัญชาติ (รายการค้นหา) ก็ได้รับการแปลด้วย
