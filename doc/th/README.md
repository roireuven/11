## ดัชนีเอกสาร

คู่มือผู้ใช้สำหรับการตั้งค่า การดำเนินงานประจำวัน การบริหาร และอ้างอิงทางเทคนิค ใช้แถบด้านข้างเพื่อเรียกดูหัวข้อ

## ขั้นตอนการสนับสนุน

1. อ่าน **เริ่มต้นใช้งาน** และ **การตั้งค่าครั้งแรก**
2. กำหนดบทบาทด้วย **บทบาทและสิทธิ์ผู้ใช้**
3. กำหนดค่าทรัพย์สินใน **การตั้งค่าและการกำหนดค่า**
