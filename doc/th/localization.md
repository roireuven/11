# การแปลภาษา

HotelRestaurantMini-MartManagement รองรับ **21 ภาษาอินเทอร์เฟซ** พร้อมเลย์เอาต์ RTL สำหรับภาษาอาหรับและฮีบรู

## โลเคลที่รองรับ

| Code | ภาษา |
|------|----------|
| `en` | English |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |
| `ja` | 日本語 |
| `ko` | 한국어 |
| `ar` | العربية |
| `hi` | हिन्दी |
| `th` | ไทย |
| `vi` | Tiếng Việt |
| `id` | Bahasa Indonesia |
| `tr` | Türkçe |
| `ru` | Русский |
| `it` | Italiano |
| `nl` | Nederlands |
| `pl` | Polski |
| `he` | עברית |
| `lo` | ລາວ |
| `pt-BR` | Português (Brasil) |
| `zh-Hans` | 简体中文 |
| `zh-Hant` | 繁體中文 |

เอกสารมีในทุกภาษา UI ของแอป หน้าที่ยังไม่แปลจะลิงก์ไปยังต้นฉบับภาษาอังกฤษ

สำหรับการเปลี่ยนภาษา RTL และตำแหน่งไฟล์โลเคล ดู [English localization guide](../en/localization.md).
