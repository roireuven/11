# Gezinme ve UI

Uygulama üst çubuk, kenar çubuğu (menü), ana içerik alanı ve mobil alt gezinme ile bir SPA'dır.

## Uygulama içi dokümantasyon

| Konum | Etiket | Eylem |
|----------|-------|--------|
| **Üst çubuk** | Dokümantasyon | Gömülü dokümantasyon sayfasını açar |
| **Hamburger menü (☰)** | Yardım → Dokümantasyon | Kenar çubuğunun üstündeki ilk bölüm |
| **Alt çubuk** | Docs | Alt gezinme (mobil) |

Düzen ayrıntıları, karanlık mod ve dil seçici için bkz. [English navigation guide](../en/navigation-and-ui.md).
