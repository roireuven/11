# HotelRestaurantMini-MartManagement — Dokümantasyon

**HotelRestaurantMini-MartManagement** (v2.0) için eksiksiz dokümantasyon — **otel**, **restoran**, **mini market** ve **POS** operasyonları için hepsi bir arada mülk yönetim sistemi.
> Bazı sayfalar yalnızca İngilizce olarak mevcuttur; bağlantılar İngilizce sürüme yönlendirir.


> **Çevrimiçi görüntüle (Firebase):** [https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=tr](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=tr)  
> **Uygulama içinde:** Kenar çubuğu → **Dokümantasyon** (tüm roller) — gömülü yardım  
> **Ayna (GitHub Pages):** [GitHub Pages doc](https://roireuven.github.io/11/doc/?lang=tr)
