# Vue d'ensemble

**HotelRestaurantMini-MartManagement** est une application de gestion unifiée pour les établissements hôteliers de petite et moyenne taille combinant hôtel, restaurant et commerce sur place.

## Modules principaux

- **Hôtel / PMS** — Chambres, réservations, check-in/out, ménage, maintenance
- **Restaurant et cuisine** — Service à table, room service, file d'attente cuisine
- **Supérette et caisse** — Ventes boutique, charges chambre, caisse
- **Administration** — Paramètres, comptes, rapports, sauvegarde, localisation

Pour la vue d'ensemble complète avec tableaux d'audience et de capacités, consultez le [English overview](https://hotel-restaurant-minimart2-4.web.app/doc/?lang=en#/overview).
