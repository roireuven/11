# התחלה

מדריך זה מוביל נכס חדש מאפס לתפעול יומי במינימום שלבים.

## התחלה מהירה

- פתחו את אפליקציית ה-Web או התקינו APK ל-Android.
- השלימו **הגדרה ראשונית** עם דוא"ל וסיסמת מנהל.
- הגדירו פרופיל מלון, מטבע, מסים ועונות ב**הגדרות**.
- הוסיפו חדרים, חשבונות צוות והקצו תפקידים.
- הדריכו צוות עם מדריכי מודולים (מלון, מסעדה, מיני-מרקט) והגדירו גיבוי לפני עלייה לאוויר.

## מדריך מלא

להוראות מפורטות שלב אחר שלב, ראו את [English getting started guide](https://hotel-restaurant-minimart.firebaseapp.com/doc/?lang=en#/getting-started).
